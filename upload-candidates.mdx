---
title: Upload Candidates
description: Add candidates to your pipeline via resume upload or spreadsheet import.
---

## PDF resume upload

1. Go to **Candidates** or open a specific job
2. Click **Upload resumes**
3. Select one or more PDF files (up to 50 at a time)
4. Talenzly parses name, email, phone, skills, experience, and education from each resume
5. Parsed candidates appear in your pipeline within seconds

## Excel / CSV import

Use a spreadsheet when you already have candidate data in a structured format.

1. Click **Upload resumes → Import from Excel/CSV**
2. Download the sample template to see the expected columns
3. Fill in your data and upload the file

<Note>
  Excel/CSV import is available on Starter plan and above.
</Note>

## Deduplication

Talenzly automatically detects duplicate candidates based on email address. If a candidate with the same email already exists in your org, the upload is skipped to avoid duplicates.

## Supported file types

| Type | Notes |
|------|-------|
| PDF  | Best results — AI parser extracts structured data |
| Excel (.xlsx) | Structured import via template |
| CSV  | Structured import via template |
